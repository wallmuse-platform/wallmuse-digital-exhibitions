# LinkedIn Post - Wallmuse Descriptions App

🎨 **Introducing the Wallmuse Descriptions App - Real-time Artwork Display System**

Just documented our latest React-based application that brings art exhibitions to life with
synchronized, real-time information display!

**Key Features:**

🖼️ **Dynamic Artwork Display** - Shows high-resolution images with detailed metadata including
title, author, datation, and rich HTML descriptions

⏱️ **Real-time Synchronization** - Automatically transitions between artworks based on precise
timing calculations, keeping multiple screens in sync

🎛️ **Interactive Navigation** - Users can manually browse with previous/next controls or jump to
specific moments using the timeline slider

🎨 **Multi-tenant Theming** - Dynamically adapts visual styling based on subdomain (Wallmuse,
ShareX, AVIFF, OOO2) for different exhibition contexts

🌐 **Screen-specific Content** - Each display can show different content based on screen parameters,
perfect for multi-room exhibitions

📱 **Responsive Design** - Built with Material-UI for consistent experience across devices

🔄 **Auto-scroll & Transitions** - Smooth animations and automatic scrolling enhance the viewing
experience

**Technical Stack:** React 18, Material-UI, Axios, XML-to-JSON processing, and internationalization
support.

Perfect for museums, galleries, and digital art installations where synchronized, informative
displays enhance visitor engagement!
